# Sptrans

O objetivo dessa gem é facilitar o acesso à [API Olho Vivo][api-olho-vivo]

## Installation

Add this line to your application's Gemfile:

    gem 'sptrans'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install sptrans
